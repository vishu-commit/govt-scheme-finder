# Government Scheme Finder

A responsive web application built using React.js that helps users search and explore government schemes through category-based filtering and a user-friendly interface.

## 🚀 Features
- Search government schemes
- Filter schemes by category
- Responsive UI
- Fast performance with Vite
- Live deployment on Vercel

## 🛠️ Technologies Used
- React.js
- JavaScript
- Tailwind CSS
- React Router DOM
- Vite
